\# 三江源生态智能问答平台



!\[项目封面](https://images.unsplash.com/photo-1559827260-dc66d52bef19?ixlib=rb-4.0.3\&auto=format\&fit=crop\&w=1950\&q=80)



\## 🌍 项目简介



三江源生态智能问答平台是一个专注于三江源国家公园生态保护知识的智能问答系统。平台通过前端（Vercel部署）与后端（Supabase数据库）的分离架构，实现用户管理、智能问答、数据统计等功能，致力于推广三江源生态保护知识。



\## ✨ 核心功能



\### 👥 用户系统

\- \*\*用户注册/登录\*\*：支持手机号注册与登录

\- \*\*管理员系统\*\*：专用管理员后台，查看用户行为分析

\- \*\*会话管理\*\*：自动记录用户在线时长和登录状态



\### 💬 智能问答

\- \*\*生态知识问答\*\*：关于三江源生态的专业问答

\- \*\*问题收藏\*\*：用户可以收藏有价值的问答记录

\- \*\*历史记录\*\*：自动保存用户的问答历史



\### 📊 数据统计

\- \*\*个人数据\*\*：用户可查看自己的登录次数、在线时长等

\- \*\*管理员统计\*\*：全局用户行为分析、周/月活跃度统计

\- \*\*可视化图表\*\*：使用Chart.js展示数据趋势



\## 🏗️ 技术架构



\### 前端技术栈

\- \*\*HTML5 + CSS3\*\*：页面结构与样式

\- \*\*Tailwind CSS\*\*：现代化CSS框架

\- \*\*JavaScript (ES6+)\*\*：交互逻辑

\- \*\*Chart.js\*\*：数据可视化

\- \*\*Font Awesome\*\*：图标库

\- \*\*Supabase JS\*\*：数据库客户端



\### 后端技术栈

\- \*\*Supabase\*\*：BaaS（后端即服务）

&nbsp; - PostgreSQL数据库

&nbsp; - 行级安全策略（RLS）

&nbsp; - 实时订阅

&nbsp; - 存储桶



\### 部署平台

\- \*\*Vercel\*\*：前端部署（免费、自动HTTPS）

\- \*\*Supabase\*\*：后端服务（免费额度充足）
