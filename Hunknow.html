GitHub 仓库文件上传操作指引

方式1：网页端直接上传文件

适用于简单文件上传，无需安装 Git 工具

  • 步骤1：打开你的 GitHub 仓库页面（如 https://github.com/你的用户名/仓库名）
  • 步骤2：找到并点击页面中的 uploading an existing file 蓝色链接
  • 步骤3：拖拽本地文件到上传区域，或点击 choose your files 选择文件
  • 步骤4：在页面底部填写提交说明（commit message），点击 Commit changes 完成上传

方式2：命令行（Git）上传文件

适用于本地已有仓库、批量文件上传场景，需先安装 Git

  • 步骤1：打开本地终端/命令提示符，进入本地仓库目录
  • 步骤2：关联远程 GitHub 仓库（首次操作）：
git remote add origin https://github.com/你的用户名/仓库名.git
  • 步骤3：将本地分支命名为 main（适配 GitHub 默认分支）：
git branch -M main
  • 步骤4：推送本地文件到 GitHub 仓库：
git push -u origin main
